# Data

- `target_composition_table.csv` — target composition descriptors (Z2, A2, I2, density, n_e)
  built in-house for 284 targets (98.3% of compound records resolved). Released under CC-BY 4.0.

## Raw experimental data (NOT included)
`StoppingPower.csv` / `StoppingPower_refs.csv` are derived from the IAEA electronic
stopping power database. **Redistribution requires permission from the IAEA.**
Obtain the data from the IAEA Nuclear Data Section and place the two CSV files in this
folder before running `src/pipeline.py`.
